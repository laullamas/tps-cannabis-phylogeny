# Terpene synthase phylogeny in *Cannabis sativa*

Genomic analysis of the TPS gene family in *Cannabis sativa* using *Arabidopsis thaliana* as reference.

## Repository contents
| File | Description |
|------|-------------|
| `tps_arath.fasta` | 33 *A. thaliana* TPS (Swiss-Prot) |
| `bona_fide.fasta` | 3 biochemically characterized anchors |
| `cannabis_tps_candidates.fasta` | 33 *C. sativa* TPS candidates (RefSeq) |
| `tps_arath_bonafide_cannabis_raw.fasta` | MAFFT input (69 seqs) |
| `tps_arath_bona_cannabis_aligned.fasta` | Multiple alignment (1272 positions) |
| `tree_tps_cannabis.nhx` | PhyML Newick tree |
| `tree_tps_cannabis_stats.txt` | PhyML statistics |
| `informe.pdf` | Full analysis report |
